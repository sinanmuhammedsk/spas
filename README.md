## 📌 Overview
An end-to-end Machine Learning pipeline and Streamlit dashboard for predicting student academic outcomes based on behavioral, demographic, and background factors. This system provides actionable insights for educators to identify at-risk students and implement targeted interventions.

## ✨ Key Features & Outcomes
- **Predictive Analytics:** Implemented robust machine learning models to forecast student success.
- **Quantifiable Accuracy:** Evaluated and optimized models using R² (R-Squared) and MSE (Mean Squared Error) metrics.
- **Interactive Dashboard:** Built a complete, user-friendly frontend using Streamlit for easy data input and result visualization.
- **Data Exploration:** Comprehensive Exploratory Data Analysis (EDA) of hidden student data patterns.

## 🏗️ Architecture & Stack
**Tech Stack:** `Python, Pandas, NumPy, Scikit-Learn, Streamlit`
- **Data Ingestion:** Reads historical student records and standardizes analytical parameters.
- **Model Training:** Automated pipeline utilizing generalized linear/tree-based algorithms.
- **Inference Engine:** Real-time feedback and dynamic predictions hosted on the Streamlit backend.

## 🚀 Installation & Usage
```bash
# Clone the repository
git clone https://github.com/sinanmuhammedsk/spas.git

# Navigate to directory
cd spas

# Run the training and frontend
streamlit run app.py
```
